### 🔄 Sorting Algorithms

#### ⚡ Quick Sort

* Recursive implementation
* Divide and Conquer approach
* Pivot-based partitioning

**Time Complexity**

* Best/Average: **O(n log n)**
* Worst: **O(n²)**

#### 📌 Insertion Sort

* In-place comparison sorting
* Efficient for small or nearly sorted arrays

**Time Complexity**

* Best: **O(n)**
* Average/Worst: **O(n²)**
